# Deploy Workflow: Claude Code Rules

*Этот файл — для AI-ассистента пользователя. AI читает его и выполняет деплой правил из ElucidatingYourLLM.*

## Контекст

Ты помогаешь пользователю установить правила Claude Code из репозитория ElucidatingYourLLM.
Правила находятся рядом с этим файлом в папке `claude-code/ru/`:

- `CLAUDE.md` — 11 основных правил (безусловная загрузка)
- `.claude/rules/*.md` — 3 правил, запускаемых по условию:
  - `file-editing.md`
  - `pre-install-check.md`
  - `structural-analogy.md`

## Шаги деплоя

### 1. Прочитай правила из этой папки

- Прочитай `CLAUDE.md` из папки `claude-code/ru/` этого репозитория.
- Составь список файлов `.claude/rules/*.md` в папке `claude-code/ru/`.

### 2. Проверь текущие правила пользователя

- Найди `CLAUDE.md` в корне рабочего проекта пользователя (если есть).
- Найди файлы в `.claude/rules/` пользователя (если есть).
- Если файлов нет — пропусти бэкап и диагностику, сразу переходи к шагу 5.

### 3. Бэкап

Если у пользователя есть существующие файлы — спроси подтверждение и сделай бэкап:

```bash
cp CLAUDE.md CLAUDE.md.backup
cp -r .claude/rules .claude/rules.backup
```

### 4. Диагностика (если у пользователя уже есть правила)

Сравни и покажи пользователю три категории:

- **Новые** — правила из ElucidatingYourLLM, которых нет у пользователя. Можно добавить безопасно.
- **Перекрывающиеся** — аналогичная тема уже покрыта у пользователя. Покажи разницу и объясни, что меняется.
- **Конфликтующие** — явное противоречие. Предложи варианты: взять новое, оставить старое, объединить.

### 5. Установка

После подтверждения пользователя:

- Скопируй `CLAUDE.md` из `claude-code/ru/` в корень рабочего проекта пользователя.
- Скопируй все `.md` файлы из `claude-code/ru/.claude/rules/` в `.claude/rules/` проекта.

Создай `.claude/rules/` если не существует.

### 6. Отчёт

Сообщи что установлено:
- 11 основных правил в `CLAUDE.md`
- 3 правил в `.claude/rules/`
- Перечисли файлы из `.claude/rules/`, которые были установлены.
