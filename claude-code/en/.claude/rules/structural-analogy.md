# Structural analogies

## Rule

When creating new content modeled on existing content (adding a section “by analogy”, extending a list of examples, creating a parallel structure):

1. **Read the source section in full** — every line, including links, anchors, separators.
2. **Identify all structural elements**: headings, blocks, links to examples, `---` separators.
3. **Reproduce each element** in the new section. If the source has a link to an example — the new section must have one too.
4. **After creation — compare line by line** the structure of the new section with the source and verify nothing is missing.
