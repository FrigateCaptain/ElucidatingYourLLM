# Deploy Workflow: OpenClaw Rules

*Этот файл — для AI-ассистента пользователя. AI читает его и выполняет деплой правил из ElucidatingYourLLM.*

## Контекст

Ты помогаешь пользователю установить правила OpenClaw из репозитория ElucidatingYourLLM.
Правила находятся рядом с этим файлом в папке `openclaw/ru/`:

- `agents-rules.md` — 11 основных правил + блок-роутер read-on-demand
- `specific/` — 3 файлов, запускаемых по условию:
  - `file-editing.md`
  - `pre-install-check.md`
  - `structural-analogy.md`

## Шаги деплоя

### 1. Прочитай правила из этой папки

- Прочитай `agents-rules.md` из папки `openclaw/ru/` этого репозитория.
- Составь список файлов в `openclaw/ru/specific/`.

### 2. Проверь текущие правила пользователя

- Прочитай `~/.openclaw/workspace/AGENTS.md` пользователя (если существует).
- Проверь наличие папки `~/.openclaw/workspace/specific/`.
- Если `AGENTS.md` не существует — пропусти бэкап и диагностику, сразу переходи к шагу 5.

### 3. Бэкап

Если у пользователя есть существующий `AGENTS.md` — спроси подтверждение и сделай бэкап:

```bash
cp ~/.openclaw/workspace/AGENTS.md ~/.openclaw/workspace/AGENTS.md.backup
```

### 4. Диагностика (если у пользователя уже есть правила)

Сравни секции `agents-rules.md` с секциями текущего `AGENTS.md`:

- **Новые секции** — не пересекаются с текущим `AGENTS.md`. Можно добавить безопасно.
- **Перекрывающиеся** — аналогичная секция уже есть. Покажи разницу и объясни, что меняется.
- **Конфликтующие** — явное противоречие. Предложи варианты: взять новое, оставить старое, объединить.

Также: есть ли в current `AGENTS.md` раздел «Specific Rules (read on demand)»? Если да — покажи разницу в списке файлов.

### 5. Установка

После подтверждения пользователя:

- Добавь новые секции из `agents-rules.md` в `~/.openclaw/workspace/AGENTS.md`.
  - Не дублируй секции, которые уже есть.
  - Для перекрывающихся и конфликтующих — следуй решению пользователя из шага 3.
- Скопируй файлы из `specific/` в `~/.openclaw/workspace/specific/`.
  - Создай папку `specific/` если не существует.
  - Для существующих файлов — спроси перед перезаписью.

### 6. Отчёт

Сообщи что установлено:
- Сколько секций добавлено в `AGENTS.md`
- 3 файлов скопировано в `specific/`
- Перечисли файлы из `specific/`, которые были установлены.
