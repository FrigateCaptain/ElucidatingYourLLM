# Cursor: установка правил

Правила для Cursor IDE из **ElucidatingYourLLM** (14 правил: 11 основных + 3 по условию).



## Что здесь

- `.cursorrules` — основные правила (загружаются автоматически при каждом диалоге)
- `.cursor/rules/*.mdc` — правила для конкретных задач (загружаются автоматически по условию: типу файла или описанию задачи)

## Установка вручную

1. Скопируйте `.cursorrules` в корень вашего проекта.
2. Скопируйте папку `.cursor/` в корень вашего проекта.
3. Перезапустите Cursor или откройте новый диалог.

Если в вашем проекте уже есть `.cursorrules` — сделайте бэкап перед копированием.

## Установка через AI

Дайте AI ссылку на файл воркфлоу:

```
Прочитай https://github.com/FrigateCaptain/ElucidatingYourLLM/blob/main/cursor/ru/deploy-workflow.md и установи правила.
```

AI проверит конфликты с вашими текущими правилами, предложит бэкап и выполнит установку.
