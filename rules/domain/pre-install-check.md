## Pre-Installation Compatibility Check [MANDATORY]

**Перед установкой любого приложения или пакета — проверить совместимость с целевой системой.**

1. **Определить требования приложения**:
   - Минимальная версия ОС
   - Требуемые библиотеки (GLIBC, libstdc++ и т.д.)
   - Архитектура (x86_64, ARM и т.д.)
   - Зависимости (runtime, frameworks)

2. **Определить параметры целевой системы**:
   - Версия ОС: `lsb_release -a` или `cat /etc/os-release`
   - Версия GLIBC: `ldd --version | head -1`
   - Архитектура: `uname -m`

3. **Сравнить и принять решение**:
   - Совместимы → приступать к установке
   - **Несовместимы** → остановиться, сообщить о конфликте и предложить альтернативы:
     - Более старая/совместимая версия
     - Контейнер (Docker, Flatpak, AppImage)
     - Обновление системы (если приемлемо)

4. **Источники информации**:
   - Страница релиза (GitHub Releases, changelog)
   - Официальная документация
   - Веб-поиск: `"[app name] [version] system requirements linux"`

**Причина**: Установка несовместимого ПО → ошибки вида `GLIBC_2.36 not found`, потеря времени.
