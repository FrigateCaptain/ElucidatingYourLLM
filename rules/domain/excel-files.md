# Excel Files — Работа с Excel через openpyxl

**Доменное правило.** Применять при работе с `.xlsx`, `.xls`.

Globs: `**/*.xlsx, **/*.xls`

---

## Excel Files Rules [MANDATORY]

При модификации существующих Excel файлов:

1. **Загружать** через `openpyxl.load_workbook()` — НИКОГДА не использовать `Workbook()`
2. **Не менять форматирование** существующих ячеек без явного запроса
3. **Изменять только указанные ячейки** — не трогать остальные
4. **Сохранять под новым именем** с добавлением суффикса версии
5. **При неуверенности** — уточнить перед изменениями

Форматирование:
- Даты — в правильном формате Excel
- Заголовки таблиц — выделять жирным
