# Generated Documents — Создание DOCX/ODT

**Доменное правило.** Применять при программном создании текстовых документов.

Globs: `**/*.docx, **/*.odt`

---

## Generated Document Files Rules [MANDATORY]

При программном создании `.docx`, `.odt`:

1. **Язык документа [MANDATORY]**: Устанавливать язык, соответствующий языку содержимого.
   - Определять автоматически (весь текст русский → `ru-RU`, английский → `en-US`)
   - Многоязычное содержимое — спросить пользователя
   - Не полагаться на язык по умолчанию библиотеки (python-docx по умолчанию → `en-US`)

2. **Для `.docx` (python-docx)**: Устанавливать язык в трёх местах:
   - `docDefaults` → `rPrDefault` → `rPr` → `w:lang`
   - `themeFontLang` в `settings`
   - Стиль `Normal` → `rPr` → `w:lang`

**Причина**: Неправильный язык → проверка орфографии подчёркивает весь текст как ошибочный.
